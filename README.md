# Python Calculator 🧮

A simple and functional desktop calculator built with Python and Tkinter.

## 🚀 Features
- Basic arithmetic operations (+, -, *, /)
- Error handling (e.g., Division by Zero)
- Clean and user-friendly interface
- Clear button to reset input

## 🛠️ Technologies Used
- Python 3
- Tkinter (GUI Library)
